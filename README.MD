## DescripciГіn general

Prescripciones MГ©dicas MVP centraliza el proceso bГЎsico de emisiГіn, consulta, consumo y descarga de recetas mГ©dicas digitales.

La aplicaciГіn estГЎ organizada en tres experiencias principales:

- **Administrador:** consulta mГ©tricas generales del sistema desde un dashboard visual.
- **MГ©dico:** revisa, filtra y pagina sus prescripciones; tambiГ©n crea nuevas recetas para pacientes existentes.
- **Paciente:** consulta sus prescripciones, filtra por estado, revisa el detalle, confirma el consumo de recetas y puede ver o descargar el PDF correspondiente.

El sistema prioriza una experiencia sobria y directa, con reglas de negocio protegidas desde backend y una interfaz web preparada para demostrar el flujo completo desde una URL pГәblica.
